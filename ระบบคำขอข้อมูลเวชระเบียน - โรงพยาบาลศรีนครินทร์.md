# ระบบคำขอข้อมูลเวชระเบียน - โรงพยาบาลศรีนครินทร์

ระบบออนไลน์สำหรับการยื่นคำร้องข้อมูลเวชระเบียนที่ปลอดภัย รวดเร็ว และเป็นไปตามพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล (PDPA)

## 📋 ภาพรวม

ระบบนี้ประกอบด้วย:

- **Frontend**: HTML5, CSS3, JavaScript (Static)
- **Backend**: Google Apps Script (Code.gs)
- **Database**: Google Sheets (9 sheets)
- **Authentication**: Firebase + Google Sign-In
- **Storage**: Google Drive (Avatar + Attachments)
- **Notifications**: In-web Polling + Email
- **Deployment**: GitHub Pages

## 🚀 การติดตั้ง

### 1. Clone Repository

```bash
git clone https://github.com/your-repo/srinagarind-medical-record-request.git
cd srinagarind-medical-record-request
```

### 2. ตั้งค่า Firebase

1. ไปที่ [Firebase Console](https://console.firebase.google.com/)
2. สร้าง Project ใหม่
3. เปิด Authentication และเลือก Google Sign-In
4. คัดลอก config ไปที่ `js/config.js`

### 3. ตั้งค่า Google Apps Script

1. ไปที่ [Google Apps Script](https://script.google.com/)
2. สร้าง Project ใหม่
3. คัดลอก `Code.gs` ลงไป
4. Deploy เป็น Web App
5. ใส่ URL ใน `js/config.js` ที่ `APPS_SCRIPT_URL`

### 4. ตั้งค่า Google Sheets

1. สร้าง Google Sheets ตามรายการด้านล่าง
2. ใส่ Spreadsheet ID ใน `js/config.js`

## 📱 ฟีเจอร์หลัก

### สำหรับผู้ใช้ทั่วไป

1. **ลงทะเบียน**: ยื่นข้อมูลส่วนบุคคล รอการอนุมัติ
2. **ตั้งค่าโปรไฟล์**: อัปโหลดรูป แก้ไขข้อมูล
3. **ยื่นคำร้อง**: เลือกผู้ป่วย เอกสาร วัตถุประสงค์
4. **ติดตามสถานะ**: ดูความคืบหน้าแบบเรียลไทม์
5. **ดาวน์โหลด**: รับข้อมูลเวชระเบียนเมื่อเสร็จสิ้น
6. **การแจ้งเตือน**: รับข้อมูลอัปเดตทันที

### สำหรับเจ้าหน้าที่

1. **อนุมัติผู้ใช้**: ตรวจสอบ อนุมัติ/ปฏิเสธ
2. **จัดการคำร้อง**: ประมวลผล อนุมัติ ปฏิเสธ
3. **ติดตาม SLA**: เตือนเมื่อใกล้หมดเวลา
4. **รายงาน**: สรุปสถิติ อัตราการปฏิบัติตาม SLA
5. **ตั้งค่า**: ปรับ SLA, ข้อมูลติดต่อ, นโยบาย

## 🐛 Troubleshooting

### ปัญหา: ไม่สามารถเข้าสู่ระบบ

**สาเหตุ**: Firebase ไม่ได้ตั้งค่า หรือ config ผิด

**วิธีแก้**:
1. ตรวจสอบ `js/config.js`
2. ตรวจสอบ Firebase Console
3. ตรวจสอบ Google Sign-In settings

### ปัญหา: ไม่ได้รับ Email

**สาเหตุ**: Email service ไม่ได้ตั้งค่า

**วิธีแก้**:
1. ตรวจสอบ `Code.gs` - EMAIL_ADDRESS
2. ตรวจสอบ Gmail settings
3. ตรวจสอบ Less Secure Apps

### ปัญหา: ข้อมูลไม่ปรากฏ

**สาเหตุ**: Google Sheets ไม่ได้ตั้งค่า

**วิธีแก้**:
1. ตรวจสอบ Spreadsheet ID
2. ตรวจสอบ Sheet names
3. ตรวจสอบ Permissions
