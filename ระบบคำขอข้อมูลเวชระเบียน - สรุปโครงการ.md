## 🎯 วัตถุประสงค์

### วัตถุประสงค์หลัก
1. **ลดเวลาการประมวลผล** - จาก 7 วัน เหลือ 3 วัน (ปกติ) / 1 วัน (เร่งด่วน)
2. **เพิ่มความปลอดภัยข้อมูล** - ปฏิบัติตาม PDPA ด้วยการเข้ารหัสและ Audit Log
3. **ปรับปรุงประสบการณ์ผู้ใช้** - ติดตามสถานะแบบเรียลไทม์
4. **ลดภาระงาน** - ระบบอัตโนมัติสำหรับการประมวลผล

### วัตถุประสงค์รอง
1. ให้เจ้าหน้าที่จัดการคำร้องอย่างมีประสิทธิภาพ
2. สร้างรายงานสถิติและ SLA
3. บันทึกการตรวจสอบ (Audit Log) ครบถ้วน

## 📊 Google Sheets Structure

ระบบใช้ 9 Google Sheets:

| Sheet | คอลัมน์ | วัตถุประสงค์ |
|-------|--------|-----------|
| **users** | userId, email, firstName, lastName, status, ... | ข้อมูลผู้ใช้ |
| **requests** | requestId, userId, status, patients, documents, ... | คำร้องข้อมูล |
| **patients** | patientId, requestId, hn, name, ... | ข้อมูลผู้ป่วย |
| **documents** | docId, value, labelTH, labelEN | เอกสารที่ขอ |
| **notifications** | notifId, userId, title, message, read, ... | การแจ้งเตือน |
| **audit_log** | logId, userId, action, timestamp, ... | บันทึกการตรวจสอบ |
| **settings** | key, value | ตั้งค่าระบบ |
| **master_data** | type, value, labelTH, labelEN | ข้อมูลอ้างอิง |
| **logs** | logId, level, message, timestamp | บันทึกข้อผิดพลาด |

## 🔐 ความปลอดภัย

### PDPA Compliance
- ✅ **Encryption**: ข้อมูลถูกเข้ารหัสแบบ End-to-End
- ✅ **Audit Log**: บันทึกทุกการเข้าถึงและการเปลี่ยนแปลง
- ✅ **Consent**: ผู้ใช้ต้องยอมรับนโยบายคุ้มครองข้อมูล
- ✅ **Data Retention**: ลบข้อมูลตามนโยบาย
- ✅ **Right to be Forgotten**: ผู้ใช้สามารถขอลบข้อมูล

### Authentication & Authorization
- ✅ **Firebase Auth**: ตรวจสอบตัวตนผ่าน Google
- ✅ **Role-based Access**: Admin, Staff, User
- ✅ **Session Management**: หมดเวลา Session อัตโนมัติ
- ✅ **Two-Factor Auth**: สำหรับ Admin (Optional)

## 📋 ขั้นตอนการติดตั้ง

### 1. ตั้งค่า Firebase
```
1. สร้าง Firebase Project
2. เปิด Authentication
3. เลือก Google Sign-In
4. คัดลอก Config ไปที่ js/config.js
```

### 2. ตั้งค่า Google Apps Script
```
1. สร้าง Google Apps Script Project
2. คัดลอก Code.gs
3. Deploy เป็น Web App
4. ใส่ URL ใน js/config.js
```

### 3. สร้าง Google Sheets
```
1. สร้าง Spreadsheet ใหม่
2. สร้าง 9 Sheets ตามรายการ
3. ใส่ Spreadsheet ID ใน js/config.js
```

### 4. Deploy ไปยัง GitHub Pages
```
1. Push ไปยัง GitHub
2. เปิด GitHub Pages
3. ตั้งค่า Custom Domain (ถ้ามี)
```
