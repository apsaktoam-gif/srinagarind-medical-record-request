## 🚀 ขั้นตอนการติดตั้ง

### 1. ตั้งค่า Firebase
```
1. สร้าง Firebase Project ที่ https://console.firebase.google.com/
2. เปิด Authentication
3. เลือก Google Sign-In
4. คัดลอก Config ไปที่ js/config.js
```

### 2. ตั้งค่า Google Apps Script
```
1. ไปที่ https://script.google.com/
2. สร้าง Project ใหม่
3. คัดลอก Code.gs ลงไป
4. Deploy เป็น Web App
5. ใส่ URL ใน js/config.js ที่ APPS_SCRIPT_URL
```

### 3. สร้าง Google Sheets
```
1. สร้าง Google Sheets ใหม่
2. สร้าง 9 Sheets ตามรายการด้านบน
3. ใส่ Spreadsheet ID ใน js/config.js
```

### 4. Deploy ไปยัง GitHub Pages
```
1. Push ไปยัง GitHub
2. เปิด GitHub Pages ใน Settings
3. ตั้งค่า Custom Domain (ถ้ามี)
```
